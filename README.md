\# 🚀 Big Data Spam Detection System with Apache Spark



An enterprise-grade, scalable spam detection system using Apache Spark, Kafka streaming, React.js, and Flask, deployed with Docker and Kubernetes.



\## ✨ Features



\- \*\*Distributed Processing\*\*: Apache Spark for horizontal scaling

\- \*\*Real-time Streaming\*\*: Kafka integration for live email processing

\- \*\*Multiple Algorithms\*\*: Random Forest, Logistic Regression, SVM, Naive Bayes

\- \*\*Ensemble Learning\*\*: Weighted voting for better accuracy

\- \*\*Interactive Dashboard\*\*: React.js with Material-UI

\- \*\*RESTful API\*\*: Flask backend with WebSocket support

\- \*\*Containerized\*\*: Docker + Kubernetes deployment

\- \*\*Scalable\*\*: Handles millions of emails
